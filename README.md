# 🩺 Disease Prediction using Machine Learning

A Machine Learning project for predicting the risk of **diabetes and heart disease** using multiple classification models.

## 📊 Models Used

- Logistic Regression  
- Support Vector Machine (SVM)  
- Random Forest  
- XGBoost  

## 📈 Model Performance (Diabetes Dataset)

| Model               | Accuracy |
|--------------------|----------|
| Logistic Regression | 75.97%   |
| SVM                 | 76.62% ✅ |
| Random Forest       | 75.32%   |
| XGBoost             | 72.08%   |

## 🏆 Best Model
**Support Vector Machine (SVM)** achieved the highest accuracy among all models.

## 🔍 Key Features

- Glucose  
- BMI  
- Age  

## ⚙️ Techniques Used

- Data Cleaning (handling missing values)
- Feature Scaling (StandardScaler)
- Model Training & Comparison
- Performance Evaluation

## 🚀 Conclusion

The model can assist in early prediction of diabetes and heart disease, helping in better healthcare decision-making.

## 📁 Dataset

- Pima Indians Diabetes Dataset  
- Heart Disease Dataset (retrieved from OpenML using Scikit-learn)
The datasets include medical attributes such as glucose level, BMI, age, blood pressure, cholesterol, and other health indicators.

## 🛠️ Technologies

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- XGBoost
